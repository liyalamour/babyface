## Local preview

```bash
cd Babyface
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080).

## Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g. `babyface-website`).
2. Push this folder:

```bash
git init
git add .
git commit -m "Initial Babyface site clone"
git branch -M main
git remote add origin git@github.com:YOUR_USERNAME/babyface-website.git
git push -u origin main
```

3. On GitHub: **Settings → Pages → Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: `main` / `/ (root)`
4. Your site will be live at `https://YOUR_USERNAME.github.io/babyface-website/`

> For a custom domain, add a `CNAME` file and configure DNS in repo Settings → Pages.

## Gallery images

### Your own photos (recommended)

Drop files into the upload folders (see `uploads/README.md`):

- `uploads/newborn/` — home / 美式新生兒寫真
- `uploads/family/` — 家庭/周歲攝影
- `uploads/wedding/` — 婚禮攝影

Then regenerate the list and commit:

```bash
python3 scripts/scan-uploads.py
```

If a folder has **any** images, that page uses **only** those files. Empty folders use the default CDN list below.

### Default list (Squarespace CDN)

`data/galleries.json` holds the original URLs. To refresh from the live site:

```bash
python3 scripts/update-galleries.py
```

To download those into `assets/gallery/` for offline use:

```bash
python3 scripts/download-gallery.py
```
