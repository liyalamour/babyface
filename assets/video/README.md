# Service page video

Place your studio introduction video here for the **我們的服務** page (`service.html`):

| File | Purpose |
|------|---------|
| `service.mp4` | Main video (required) |
| `service-poster.jpg` | Optional thumbnail shown before play |

## Tips

- Use **MP4 (H.264)** for broad phone and browser support.
- Keep the file **under ~50 MB** if possible (GitHub warns above 100 MB per file).
- Compress with HandBrake or similar before uploading if the export is large.
- Optional poster: export a JPG frame (e.g. 1280×720) as `service-poster.jpg`.

After adding files, commit and push to GitHub for the live site to update.
